Modalidades

Nosso modulo especial foi estudado carinhosamente para garantir que cada criança possa exercitar suas habilidades em um ambiente seguro, com alta concentração, autoestima e disciplina. O Jiu Jitsu é uma ferramenta incrível para a evolução de uma criança; focamos no físico, mental e na saúde emocional durante os exercícios para que todos possam aproveitar.

Nosso objetivo é deixar sua entrada no Jiu Jitsu fácil para que você aproveite toda aula. Nossas aulas iniciantes foram criadas para que você tenha uma base antes de avançar para os próximos módulos.

Nosso serviço VIP, com aulas especialmente montadas para suas necessidades. Na aula particular você encontrará a aula perfeita para você, seja qual for seu objetivo dentro do esporte.

Aprendiz de alguns dos melhores coaches no mundo, nesse modulo vamos conhecer múltiplas combinações de movimentos e posições, assim você aprenderá aplicar as técnicas mais avançadas no seu Jiu Jitsu.
